# Blundell Six-Part Guide

Based on William E. Blundell, *The Art and Craft of Feature Writing* (1988). The six-part guide is Blundell's core reporting framework — a set of ordered questions that forces consideration of every important dimension of a story before and after reporting.

The six parts are not equal. In most stories, one or two dominate. The guide's value is in making you *choose* which matter for this story rather than defaulting to whatever is easiest to report.

## When to use

- Planning what to report before going into the field
- Auditing existing notes or a draft for reporting gaps
- Deciding which aspects of a story deserve the most reporting time
- Checking whether a pitch covers the right ground

## When not to use

- The story isn't scoped yet → use `blundell-cause-effect-map` first
- The question is what the story *is* (theme statement) → use `blundell-theme-statement`
- The reporting is done and you're organizing material → use `blundell-indexing`
- The question is how to structure the story → use `blundell-narrative-lines`
- The question is about writing the lead → use `blundell-leads`

## Two modes

**Pre-reporting (planning)**: Ask which of the six parts matters most for this particular story, given its stage (juvenile or mature) and the reporter's access. Return a weighted priority list with specific questions to answer for each section.

**Post-reporting (diagnostic)**: Assess existing material against each category. Return a one-paragraph assessment per category: what's covered, what's thin, what's missing.

Determine mode from context. If the reporter has material, run diagnostic. If they're planning, run planning. If unclear, ask: "Do you have reporting in hand, or are you still figuring out what to go get?"

## Non-negotiables

- The six parts are a guide, not a checklist. Don't run through all six mechanically if only two matter. Toss out or note as minor the ones that clearly don't apply to this story.
- Scope is not just scale. It has four dimensions: quantitative, locale, diversity/intensity, and perspective (what other developments magnify or temper this one). Easy to cover only the first.
- Reasons is the most neglected category — especially psychological/emotional motives. When an action doesn't make sense on its face, look for personality or ego driving it.
- Countermoves: talk is cheap, action precious. Weight what is being *done* far above what is being *said*. A countermoves section full of critics and advocates but no actual action is thin.
- Impacts requires asking how people *feel*, not just what they *think*. The emotional response of those affected is absent from most stories that should have it.
- For profiles, use the profile variant in `references/profile-variant.md`. Qualities replaces Scope; Values/Standards replaces Reasons.

## Output format

**Pre-reporting**: Weighted section list — which 1-2 sections should dominate, which get brief attention, which can be skipped — with specific questions to answer for each weighted section.

**Post-reporting**: One paragraph per active category assessing coverage quality, followed by a prioritized gap list (3-5 items, dealbreakers first).
